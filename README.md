1. 프로젝트 이름 또는 가칭
구구 (GUGU)
초등학교 고학년을 대상으로 하는 직관적이고 심플한 네이밍이며, 카훗(Kahoot!) 스타일의 빠르고 간편한 진입성을 내포하고 있습니다.

2. 프로젝트 개요
초등 고학년(3~5학년)을 위한 놀이형·맞춤형 고난도 구구단 학습 서비스입니다.
기존 유아용 서비스의 유치함을 탈피한 심플한 UI/UX를 제공하며, 교사의 문제 출제 및 채점 번거로움을 해소하고 학생에게는 몰입감 있는 학습 환경을 제공합니다.

3. 주 고객
초등학교 3~5학년 학생: 11단~41단 소수 구구단 및 두 자릿수 곱셈 등 고난도 연산 능력을 재미있게 기르고 싶은 학습자입니다.
초등학교 교사: 학급 및 학생별 연산 성적(정답률, 풀이 시간)을 효율적으로 관리하고 학습 리포트를 제공받고자 하는 지도자입니다.

4. 주요 기능
교사용 클래스 관리: 간편한 자동 학번 생성, 수업 코드/QR 발급 기능 및 학생별 정답률·풀이 시간이 포함된 대시보드 리포트를 제공합니다.
학생용 간편 접속 및 학습: 별도 가입 없이 일회성 코드만으로 접속하여 11단~41단 소수 구구단 및 두 자릿수 곱셈을 학습합니다.
동기부여 및 피드백 시스템: 실시간 타이머 기반의 클래스 내 랭킹 시스템을 운영하며, 즉각적인 오답 노트와 정답 확인 피드백을 제공합니다.
